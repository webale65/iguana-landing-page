# 🦎 Iguana Landing Page

Proyecto académico desarrollado con HTML5 y CSS3 como práctica de maquetación web y diseño responsive.

## Descripción

Iguana Landing Page es una página web informativa enfocada en la presentación de contenido visual y descriptivo sobre las iguanas. El proyecto fue desarrollado aplicando buenas prácticas de estructura semántica, estilos modernos y adaptación a diferentes dispositivos.

## Características

* Estructura semántica con HTML5
* Diseño responsive para dispositivos móviles y escritorio
* Maquetación utilizando Flexbox
* Galería de imágenes interactiva
* Efectos hover con CSS
* Enlaces externos informativos
* Organización modular del código

## Tecnologías Utilizadas

* HTML5
* CSS3
* Flexbox
* Git
* GitHub

## Objetivos del Proyecto

* Practicar maquetación web con HTML y CSS
* Aplicar diseño responsive
* Mejorar la organización de archivos y estilos
* Publicar proyectos utilizando Git y GitHub
